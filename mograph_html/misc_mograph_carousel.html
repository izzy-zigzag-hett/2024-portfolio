"Millipedes" Animated Poster Series (2023):


This video is an introduction for an imagined TV show called "Bedtime Heroes." It was created entirely in Adobe After Effects with some Adobe Illustrator.

I used this project as an opportunity to practice simple vector animations and some typography.

Goals:

This video was created as a final project in a RISD graphic design class focused on motion graphics. My goals were to practice:

  1. Typographic animation
  2. Path morphing
  3. Simple character/facial animation
  4. Working quickly (4 week timeline)
  5. Storyboarding and timing
  6. Post-animation effects (glow, transitions, etc)
